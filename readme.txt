== Description ==

Adds call and map buttons on mobile, with customizable options from the admin panel.

This plugin allows you to display a "Call" button that will directly initiate a phone call when clicked, and a "Map" button that will open Google Maps with your desired address, both on mobile devices. 

The plugin provides an admin panel for you to customize the appearance and text of the buttons, including background and text colors, as well as button size.

== Installation ==

1. Upload the `call-map-buttons` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Settings' menu and click on 'Call & Map Buttons' to configure the plugin.
4. Enter your phone number, map link, button texts, and customize the button appearance.

== Frequently Asked Questions ==

= How do I add my phone number? =
You can enter your phone number in the plugin settings, and it will appear as a clickable "Call" button on your mobile website.

= How do I change the map link? =
You can specify your map link in the plugin settings. This will link to your desired location on Google Maps.

= Can I change the colors of the buttons? =
Yes, you can customize the background and text colors for both the "Call" and "Map" buttons in the settings.
